设置

基准指数、目标配置比例、图表偏好与隐私设置

基准指数配置

设定对标基准指数，系统自动计算基准收益率并对比总收益率。

(3850 - 3500) / 3500 × 100% = 10.00%

目标配置比例

设定各资产大类的目标比例，总和应为 100%。偏离超过 ±5% 将橙色提醒。

总和: 100%

图表偏好

当前默认使用红涨绿跌，符合国内投资软件习惯

隐私与安全

开启后所有金额数字模糊显示，方便截图分享或演示

所有资产数据仅在本地浏览器中处理，不上传至任何外部服务器。上传的 Excel 文件自动保存至本地 data/uploaded/ 目录。FundLens 不会要求您输入支付宝或同花顺的账号密码。

关于 FundLens

版本: v1.0

定位: 个人资产季度快照分析工具

数据来源: 支付宝 + 同花顺 Excel 导出

技术栈: HTML / CSS / JavaScript（原型）
